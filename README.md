## Getting Started

### Prerequisites

-   Java 21+
-   Maven 3.8+
-   Spring Boot 3+

### Running the Application

``` bash
mvn spring-boot:run
```

### Accessing the API

Once the app is running, you can access the API at:

    http://localhost:8080/api
